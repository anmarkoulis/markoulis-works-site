# Markoulis Works Site
This repository contains the static website for Markoulis Works, fulfilling the legal requirements for a Greek private company (IKE).

## Structure
- `index.html`: English version of the website.
- `index_gr.html`: Greek version of the website.
- `styles.css`: CSS file for styling.

## Requirements
- Full names and addresses of all partners.
- Type/category of contribution of each partner.
- Administrator's name.
- Basic company information (Legal name, registered address, G.E.MI number, VAT number, contact details).
- Annual financial statements (when applicable).
